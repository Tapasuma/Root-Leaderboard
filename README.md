Root Leaderboard was created to track elo for players and factions. Currently supports base factions. Works in terminal

- main: executes the script
- functions: home of functions called in main
- data: stores the players, factions and elo between uses. Placeholder names can be changed through the script or directly by modifying the json file
- requirments: use _pip install -r requirements.txt_ in terminal to install dependencies

The script currently has the following options:
1. View Leaderboard
2. Change elo
3. Add new player
4. Remove player
5. Exit
